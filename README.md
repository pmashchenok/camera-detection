# camera-detection
В проекте реализована детекция людей с уличной камеры видеонаблюдения с использованием модели YOLOv8.
Система принимает видеопоток или видеофайл, обрабатывает каждый кадр с помощью предварительно обученной модели, обнаруживает объекты класса "person" и выводит на видео рамки вокруг людей с указанием вероятности обнаружения. Проект может быть использован для систем видеонаблюдения, анализа посетителей, подсчёта людей и других задач, связанных с мониторингом городской среды.
